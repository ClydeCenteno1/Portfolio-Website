## ✨ Features

- **Sticky navigation bar** with logo, nav links, and a search input
- **Sidebar** (desktop only) with quick-access section links
- **Hero section** with a welcome message, CTA buttons, and profile image
- **Featured Projects** grid — card-based layout with project thumbnails, tech tags, and creator info
- **Profile / About section** with bio, skill tags, and action buttons (Contact Me & Download Resume)
- Fully **responsive** layout using Bootstrap 5 grid

## 🚀 Getting Started

No build tools required — this is a pure static site.

```bash
# 1. Clone the repository
git clone https://github.com//portfolio-website.git

# 2. Open in your browser
cd portfolio-website
open index.html
```

Or just drag `index.html` into your browser.
